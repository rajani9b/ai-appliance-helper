# ai-appliance-helper
A camera-powered AI assistant that identifies appliances and provides simple troubleshooting, maintenance, and safety guidance assistance to users

## Inspiration

Everyday appliances often have confusing controls, error codes, and maintenance requirements. Finding the correct manual or waiting for technical support can take considerable time.

AI Appliance Helper was inspired by real experiences using a Samsung washer, Hamilton Beach freezer, and the controls in a 2012 vehicle, including researching options for upgrading the dashboard to CarPlay.
